## Reporting a Vulnerability

If you discover a security vulnerability within Ceylon Sentinel AI, please send an e-mail to the security team at [bamunuge2002@gmail.com](mailto:bamunuge2002@gmail.com). All security vulnerabilities will be promptly addressed.
